**************************************************************
Windows 64-bit:

Required software:
Microsoft Visual Studio 2017 or newer with installed Visual C++
Cygwin
Mono for Windows

Build steps:

1) Build Mono from release package with this instruction:
https://www.mono-project.com/docs/compiling-mono/windows/

2) Download Mono release package from Mono's release GitHub page. NOTE: Do
not confuse Tarball release packages and GitHub release packages! Tarball package
from Mono's website is needed to build .NET BCL library while GitHub package is needed
to build native parts of Mono with Visual C++.

3) Unzip GitHub package and open mono.sln in msvc foldder with Visual Studio.

4) Build solution with configuration "Release x64".

5) Copy file libmono-static-sgen.lib to GmodDotNet/NativeModule/NativeModule folder.

6) Copy mono/msvc/include/mono folder to GmodDotNet/NativeModule/NativeModule/includes folder.

7) Get Garry's mod header files from https://github.com/GlebChili/gmod-module-base and copy them
to GmodDotNet/NativeModule/NativeModule/includes

8) Build GmodDotNet/NativeModule/NativeModule.sln with configuration "Release x64".

9) Copy resulting gmsv_gmodnet_win64.dll to [Your Garry's Mod location]/garrysmod/lua/bin folder.

10) Create lib folder in your [Your Garry's Mod location]/garrysmod/lua/bin folder.

11) Copy Mono BCL library built on step 1 to [Your Garry's Mod location]/garrysmod/lua/bin/lib.

12) Build GmodDotNet/Gmod.NET/Gmod.NET.sln with configuration "Release x64"

13) Copy resulting Gmod.NET.dll to [Your Garry's Mod location]/garrysmod/lua/bin folder.

Linux 32-bit
Required software:
g++-8 or newer
Mono (development package)

*All build instructions assume that Your linux distributive is 32-bit itself.*

Build steps:

1) Build Mono from release package with this instruction:
https://www.mono-project.com/docs/compiling-mono/linux/

2) Copy file libmonosgen-2.0.a to GmodDotNet/NativeModule/NativeModule folder and rename it to libmonosgen-2.0-x32.a.

3) Copy mono header files to GmodDotNet/NativeModule/NativeModule/linux_includes

4) Get Garry's mod header files from https://github.com/GlebChili/gmod-module-base and copy them
to GmodDotNet/NativeModule/NativeModule/linux_includes

5) cd to GmodDotNet/NativeModule/NativeModule

6) Run g++-8:
g++-8 main.cpp -o gmsv_gmodnet_linux.dll -I linux_includes -L -l libmonosgen-2.0-x32.a -lm -lrt -ldl -lpthread -shared -static -fPIC

7) Copy resulting gmsv_gmodnet_linux.dll to [Your Garry's Mod location]/garrysmod/lua/bin folder.

8) Create lib folder in your [Your Garry's Mod location]/garrysmod/lua/bin folder.

9) Copy Mono BCL library built on step 1 to [Your Garry's Mod location]/garrysmod/lua/bin/lib.

10) Build GmodDotNet/Gmod.NET/Gmod.NET.sln with msbuild with configuration "Release Any CPU"

11) Copy resulting Gmod.NET.dll to [Your Garry's Mod location]/garrysmod/lua/bin folder.
